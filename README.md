# C-like Compiler

A Python implementation of a Scanner and Recursive Descent Parser for a C-like programming language.

## Directory Structure
- `src/`: Source code (`scanner.py`, `parser.py`).
- `tests/`: Unit tests (`test_parser.py`).
- `docs/`: Documentation and grammar.
- `logs/`: Test execution logs.

## Usage

### Prerequisites
- Python 3.x

### Run Parser
```bash
python src/parser.py
```
Type your code, and press Enter on an empty line to parse.

### Run Tests
```bash
python tests/test_parser.py
```
Check `logs/test_log.txt` for detailed results.
